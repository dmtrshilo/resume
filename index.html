Виртуальная таблица

Высокопроизводительная таблица, построенная на VDOM. Способна отрисовать огромное количество данных, без пагинации и подгрузок (на примере 500_000 записей).

Css Editor

Кастомный редактор CSS - позволяет блокировать типы селекторов или свойства с возможностью добавлять свои алиасы на селектор. Реализован как валидатор, так и хайлайтер

Чат

Чат модуль, обладает всем функционала обычного чата. Построен с применением VDOM. Работает на протоколе WebSocket

Соединительные линии

Позволяет строить линии между объектами с гибкой кастомизацей. 3 режима построения, градиенты, тексты

Grid редактор

Создает любую схему с кастомными гэпами. Доступно добавление строк и колонок, объединение ячеек гридов

Masonry

Модуль позволяет распределять объекты внутри контейнера по приницу "кирпичной кладки" с возможностью обхода препятсвий
